선언적 함수/ 함수 선언식(Function Declarations)
  • function 키워드를 사용하고 함수명 정의 후 함수 선언
  • 정의한 함수명으로 호출
  • 호출문이 먼저 나와도 에러 없이 처리
  • return값은 하나만 가능, 또는 생략 가능
    익명 함수/ 함수 표현식(Function Expressions)
  • 호출문을 함수 선언 이후에 작성해야 에러가 발생하지 않음
  • 함수를 변수에 넣고 변수로 함수를 호출한다.
내장함수
내장 함수 설명
encodeURI() 특수문자 제외한 문자만 인코딩(ex 인터넷 주소)
decodeURI() encodeURI()로 인코딩된 데이터 되돌림
isNaN() Not a Number, 숫자가 아닌 문자가 포함되면 true로 반환
isFinite 값이 유한수인지 판단
parseInt() 문자를 정수형으로 변환
parseFloat() 문자를 실수형으로
eval() 문자로 된 수식을 JavaScript의 수식으로 인식하여 실행시키고 결과 반환
Number() 문자를 숫자형으로
String() 숫자를 문자형으로
focus() 특정 요소에 포커스를 설정하거나, 페이지 로드 시 특정 필드에 자동으로 포커스를 주는 데 활용

URI(Uniform Resource Identifier)
인터넷 상의 자원을 식별하는 문자열
웹 상에서 자원의 위치를 나타내는 URL과 이름으로 식별하는 URN 등이 URI의 하위 개념

JSON(JavaScript Script Object Notation)
데이터를 교환하고 저장하기 위한 텍스트 기반의 형식

이벤트 주로 마우스나 키보드를 사용할 때, 웹 문서를 불러올 때, 폼(form)에 내용을 입력할 때 이벤트를 감지
이벤트 : 버튼을 클릭하거나 선택하는 것
이벤트 처리기 : 이벤트가 발생했을 때 실행하는 함수
주로 마우스나 키보드를 사용할 때, 웹 문서를 불러올 때, 폼(form)에 내용을 입력할 때 발생
*onclick 속성 : 클릭되었을 시 발생되는 이벤트

parsing
특정 형식으로 구성된 데이터를 분석하고 그 의미를 이해하는 과정
주로 텍스트 기반 데이터를 해석하거나, 프로그래밍 언어의 소스 코드를 이해하거나, 문서를 구조화하고 내용을 추출하는 데 사용
