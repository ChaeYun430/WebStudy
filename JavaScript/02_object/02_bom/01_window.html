특징 -global object(전역 개체)
-브라우저 객체 모델의 최상위 객체
-window의 형태와 위치를 변경할 수 있는 메서드
-생략 가능
-open() : 새로운 윈도우 객체 생성
-alert() : 경고창
-prompt() : 입력창
-confirm() : 확인/취소 창
-setInterval() : 일정 시간마다 실행
-setTimeout() : 타이머의 시간이 만료된 후 실행문을 수행
-moveTo(x, y) : 윈도우를 현재 위치에서 절대적 위치로 이동
-resizeTo(x, y) : 윈도우의 화면을 현재 크기에서 (x, y)만큼 증가시킴
-scrollBy(x, y) : 윈도우 스크롤의 위치를 절대 위치로 이동
-focus() : 윈도우에 초점을 맞춤
-blur() : 윈도우에 초점을 제거
-close() : 윈도우를 닫음
