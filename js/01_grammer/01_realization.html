스크립트
컴파일되지 않고 애플리케이션 실행되는 동안 줄 단위로 해석되는 명령어나 문장들의 집합

  • 코어(ECMAScript)
    • 프로그램 구현 언어(타입, 연산자, 선언문, 키워드, 예약어, 객체 등)로 2008년부터 주요 브라우저는 모두 ECMA-262를 준수
  • 문서 객체 모델(DOM)
    • DOM API를 이용하여 노드의 추가, 삭제, 수정이 가능하며 계속적으로 확장 지원 중
  • 브라우저 객체 모델(BOM)
    • 브라우저에 접근하고 조작할 수 있도록 지원
    JavaScript 삽입 방법
  1. HEAD의 script내에 작성
  2. BODY의 script내에 작성
  3. 외부 파일로 작성 후 script src = "" 로 HEAD에서 불러오기
    jQuery, jQuery Mobile, Sencha Touch 등의 라이브러리 언어 사용 가능
    *페이지 내 다른 객체와 어울려 작동할 경우와 HEAD에서는 작동하지 않는 경우가 있기 때문에 BODY의 어떤 곳에도 작성 가능
    주의 사항 1. JavaScript는 대소문자 구분
    2. 라인 구분 enter로 인식, 원칙적으로 세미콜론, 한 줄에 2개 이상의 문장이 기술되면 생략 불가
    -type = "text/javascript" 생략 가능
    -주석처리는 자바와 동일
