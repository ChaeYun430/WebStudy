GIF(Graphic Interchange Format)
JPEG, PNG에 비하여 파일의 크기가 작다.
표현 가능 색상이 256개 이므로 자연스러운 장면에는 부적합하다.
사진이 아닌 클립아트나 드로잉 같은 종류의 이미지에 적합하다.

JPEG(Jooint PhotoGraphic Experts Group)
24비트의 칼라를 사용하므로 수백만 개의 색상 표현 가능
복잡한 그림이나 사진 등 색상을 많이 사용하는 이미지에 적합

PNG(Portable Network Graphic)
GIF와 JPEG 형식의 장점과 비압축 형식인 BMP 형식의 장점을 가짐
24비트(또는 32비트) 컬러를 사용함

형식: img src="" width="" height="" alt="" 태그
src = "이미지 파일명 또는 웹 상의 이미지 경로"
alt = "이미지가 없을 때 출력하는 메시지"

figure 태그 그림, 사진, 다이어그램과 텍스트의 콘텐츠를 함께 묶어서 어떤 요소들을 그룹 지을 때 사용 figcaption 태그 figure태그의 요소를 위한 제목을 표현 HTML지원X: align, border hspace, vspace HTML사용X: width, height (스타일 시트에서 설정 가능)
